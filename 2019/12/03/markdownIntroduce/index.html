初识 Markdown

发表于 2019-12-03 | 分类于 Markdown

Markdown是一种轻量级的「标记语言」，通过简单的语法就能实现「易读易写」的文章，使我们能专注于文章本身的内容，而不是排版样式。

使用 Markdown 的优点

  • 语法简单，学习成本低
  • 能专注于写作，顺利思路，从排版中解脱
  • 能轻松的导出为其他发布格式

基本语法

0. 通用知识

行级语法，要用空格或Tab于文件进行间隔，语法才会生效。

1. 六级标题

在文字前加#即可定义标题；总共六级标题；符号和文字间要加空格；

2. 引用

引用别处的一小段句子，就可使用>实现。

3. 列表

无序列表的语法符号还有很多，例*或+等都行，效果是一样的。

有序列表的序号数字不必是连续的，例1.1.3.5.等只要是数字就行，最终会被自动显示为有序的。

4. 图片和超链接

区别仅在于有无!。

5. 粗体和斜体

6. 代码框

7. 表格

表格是 Markdown 的弱项，:用于指定列的对齐方式。

8. 分割符
